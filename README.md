# IS446-Managing-Customer-Relationship-With-Analytics
📊 **SCIS Track Selection Analysis Project**

📌 **Overview**

This project is based on survey data collected from 80 SCIS students. We are a group of students from SMU SCIS currently working on a project for IS446 (Managing Customer Relationships with Analytics).

The purpose of this survey is to better understand students’ experiences when choosing their specialisation track, particularly in terms of guidance and decision-making.

⚙️ **Data Processing**

Survey responses were collected from 80 SCIS students
Data cleansing was performed on the dataset, including:
- Removal of stop words
- Text normalization
- Lemmatization of open-ended responses
The cleaned dataset was exported into .xlsx format for further use

📊 **Data Preparation & Visualization**

The cleaned dataset was further processed using Tableau Prep (e.g., column splitting and pivoting)
The final dataset was used to build dashboards in Tableau for analysis and insights generation
Note: Tableau Prep and Tableau files are not included in this repository

💡 **Solution Prototype**

A simulation prototype was created using Figma to demonstrate our proposed solution:

👉 https://menu-enamel-77953257.figma.site/

This allows SCIS students to:

- Try out the proposed solution
- Better understand different specialisation tracks
- Make more informed decisions based on their interests and preferences

🎯 **Project Objective**

To improve the track selection experience by providing clearer guidance, better insights, and a more interactive decision-making process for students.
